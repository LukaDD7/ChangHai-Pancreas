# 胰腺疾病精准分型诊断临床指南调研报告

**报告日期**: 2026-03-30
**调研范围**: 实性肿瘤、囊性肿瘤、急慢性胰腺炎
**指南来源**: NCCN 2025、Atlanta 2024、IAP/APA/EPC 国际共识、中国指南 2024-2025

---

## 一、实性肿瘤

### 1.1 胰腺导管腺癌 (PDAC)

#### NCCN 2025 诊断标准

| 评估维度 | 诊断标准 | 临床意义 |
|----------|----------|----------|
| **肿瘤大小** | CT测量最大径，精确到mm | <2cm：早期；2-4cm：中期；>4cm：局部进展 |
| **血管侵犯** | 与SMA/SMV/CA/CHA接触角度 | ≤180°：BRPC；>180°：LAPC |
| **CT值** | 动脉期/静脉期双期相评估 | 等密度肿瘤需窄窗(W:150)增强检测 |
| **可切除性** | NCCN三分法 | Resectable/BRPC/LAPC |

#### 关键血管评估 (Pan-Agent核心任务)

| 血管 | 侵犯定义 | 手术影响 |
|------|----------|----------|
| **SMA** | 接触>180°或包绕 | 需血管重建或新辅助化疗 |
| **SMV** | 节段性狭窄/闭塞但可重建 | 静脉切除重建可行 |
| **CA** | 任何接触均需评估 | 影响Whipple手术决策 |
| **CHA** | 接触角度>180° | 需动脉切除重建 |
| **PV** | 节段性侵犯≤2cm | 可切除重建 |

#### 预后因素 (CT量化)

| 指标 | 阈值 | 预后意义 |
|------|------|----------|
| 肿瘤直径 | >3cm vs ≤3cm | 生存期差异显著 |
| 血管侵犯数量 | 0/1/2+根 | 多根侵犯预后差 |
| CA19-9 | >1000 U/mL | 提示隐匿转移 |
| 淋巴结转移 | 短径>1cm | N分期关键 |

---

### 1.2 胰腺神经内分泌肿瘤 (pNET)

#### WHO 2024 分级标准

| 分级 | Ki-67指数 | 核分裂象 | 分化程度 | CT特征 |
|------|-----------|----------|----------|--------|
| **G1** | <3% | <2/10HPF | 高分化 | 边界清晰，动脉期明显强化 |
| **G2** | 3-20% | 2-20/10HPF | 中分化 | 强化程度中等 |
| **G3** | >20% | >20/10HPF | 低分化 | 强化不均匀，可有坏死 |
| **G3 NET** | >20% | 高 | 高分化但高增殖 | 需与NEC鉴别 |

#### 功能性pNET诊断标准

| 类型 | 激素 | 临床症状 | CT定位 |
|------|------|----------|--------|
| **胰岛素瘤** | 胰岛素↑ | Whipple三联征 | 小病灶(<2cm)，强化明显 |
| **胃泌素瘤** | 胃泌素↑ | Zollinger-Ellison综合征 | 十二指肠/胰腺头部 |
| **胰高血糖素瘤** | 胰高血糖素↑ | 糖尿病+皮疹 | 通常>3cm |
| **VIP瘤** | VIP↑ | Verner-Morrison综合征 | 体尾部多见 |
| **无功能pNET** | 无 | 压迫症状 | 体积大，发现晚 |

#### pNET CT量化指标

| 指标 | 测量方法 | 临床意义 |
|------|----------|----------|
| 肿瘤大小 | 最大径mm | <2cm可观察；≥2cm建议切除 |
| 强化模式 | 动脉期CT值-平扫CT值 | 富血供vs乏血供 |
| 囊变比例 | 囊性成分百分比 | >50%提示恶性可能 |
| 钙化 | 有无 | 提示长期病变 |
| 血管侵犯 | 同PDAC标准 | 影响可切除性 |

## 二、囊性肿瘤

### 2.1 浆液性囊腺瘤 (SCN)

#### 诊断标准 (中国指南 2022)

| 特征 | CT表现 | 预后 |
|------|--------|------|
| **好发人群** | 中老年女性(祖母瘤) | 良性，极少恶变 |
| **形态** | 分叶状，多囊(>6个小囊) | 生长缓慢 |
| **囊壁** | 薄壁(<2mm)，无壁结节 | 手术指征严格 |
| **钙化** | 中央瘢痕伴钙化(33%) | 特征性表现 |
| **强化** | 囊壁/分隔轻度强化 | 无强化结节 |

#### 手术指征

| 指征 | 阈值 |
|------|------|
| 诊断不明确 | 无法排除恶性 |
| 症状明显 | 压迫症状/疼痛 |
| 肿瘤大小 | >6cm 或 >4cm且生长>2cm/年 |
| 侵袭表现 | 包膜不完整 |

---

### 2.2 黏液性囊腺瘤 (MCN)

#### 诊断标准

| 特征 | CT/MRI表现 | 恶性风险 |
|------|------------|----------|
| **好发人群** | 中年女性 | 5-20%恶变率 |
| **位置** | 胰体尾部 | 95%以上 |
| **形态** | 类圆形，厚壁，单房/少囊 | 囊>4cm需警惕 |
| **壁结节** | 增强壁结节≥5mm | 高度提示恶性 |
| **囊内分隔** | "囊内囊"表现 | 特征性 |

#### 手术指征 (恶性高危因素)

| 高危征象 | 阈值 | 处理 |
|----------|------|------|
| 肿瘤直径 | ≥4cm | **手术指征** |
| 壁结节 | ≥5mm且强化 | **手术指征** |
| 主胰管扩张 | ≥10mm | **手术指征** |
| 梗阻性黄疸 | 存在 | **手术指征** |
| CA19-9升高 | >37 U/mL | 结合影像评估 |

---

### 2.3 导管内乳头状黏液性肿瘤 (IPMN)

#### 分型标准

| 类型 | CT表现 | 恶变风险 | 处理策略 |
|------|--------|----------|----------|
| **主胰管型(MD)** | 主胰管弥漫性扩张(≥10mm) | 60-70% | **手术** |
| **分支胰管型(BD)** | 杵状指样囊性灶，与胰管相通 | 2.8-10.7% | 随访/手术 |
| **混合型** | 兼有两者表现 | 70%+ | **手术** |

#### BD-IPMN 手术指征 (高危征象)

| 征象 | 阈值 | 证据等级 |
|------|------|----------|
| 肿瘤直径 | >3cm | 强推荐 |
| 壁结节 | ≥5mm且强化 | 强推荐 |
| 主胰管扩张 | ≥10mm | 强推荐 |
| 梗阻性黄疸 | 存在 | 强推荐 |
| 血清CA19-9 | >37 U/mL | 中等推荐 |
| 增长速度 | >2cm/年 | 中等推荐 |

#### IPMN随访策略

| 风险分级 | CT随访间隔 |
|----------|------------|
| 低风险 | 6-12个月 |
| 中等风险 | 3-6个月 |
| 高危征象 | 立即手术 |

---

### 2.4 实性假乳头状瘤 (SPN)

#### CT诊断标准

| 特征 | CT表现 | 预后 |
|------|--------|------|
| **好发人群** | 青年女性(20-40岁) | 低度恶性 |
| **位置** | 胰体尾部多见 | 可治愈 |
| **形态** | 较大类圆形(通常>5cm) | 包膜完整预后好 |
| **成分** | 囊实性混合 | 出血坏死常见 |
| **强化** | 轻度强化，延迟强化 | 乏血供肿瘤 |

#### 手术指征

**所有SPN均应手术切除** (国内外指南一致推荐)

| 高危因素 | 提示 |
|----------|------|
| 包膜不完整 | 侵袭性增强 |
| 直径>6cm | 恶性风险增加 |
| 胰尾部位置 | 远处转移风险 |

## 三、急性胰腺炎

### 3.1 诊断标准 (Atlanta 2024)

**需满足以下3项中的2项：**

| 标准 | 内容 |
|------|------|
| 1 | 特征性上腹痛(急性发作) |
| 2 | 血清淀粉酶/脂肪酶≥正常值3倍 |
| 3 | CT/MRI示胰腺水肿/坏死/胰周渗出 |

### 3.2 严重程度分级

| 分级 | 定义 | CT表现 | 死亡率 |
|------|------|--------|--------|
| **MAP** | 无器官衰竭，无局部并发症 | 胰腺水肿，无坏死 | <1% |
| **MSAP** | 一过性器官衰竭(<48h)或有局部并发症 | 急性胰周液体积聚(APFC) | <5% |
| **SAP** | 持续性器官衰竭(>48h) | 胰腺/胰周坏死，感染性坏死 | 15-30% |
| **CAP** | SAP+感染性坏死+持续器官衰竭 | 广泛坏死，气体征 | >50% |

### 3.3 影像分型 (CT)

| 类型 | CT表现 | 处理 |
|------|--------|------|
| **间质水肿型** | 胰腺增大，强化均匀，无坏死 | 保守治疗 |
| **坏死型** | 胰腺实质无强化(>30% vs <30%) | 可能需引流/清创 |

### 3.4 CT严重度指数 (CTSI)

| 评分项 | 分值 |
|--------|------|
| **Balthazar分级** | |
| A - 正常胰腺 | 0分 |
| B - 局限性胰腺肿大 | 1分 |
| C - 胰周脂肪炎症 | 2分 |
| D - 单发液体积聚 | 3分 |
| E - 多发液体积聚/坏死 | 4分 |
| **坏死范围加分** | |
| 无坏死 | 0分 |
| <30% | 2分 |
| 30-50% | 4分 |
| >50% | 6分 |

**总分0-10分：**
- 0-3分：轻度
- 4-6分：中度
- 7-10分：重度

### 3.5 并发症定义 (修订版Atlanta)

| 并发症 | 定义 | CT表现 |
|--------|------|--------|
| **APFC** | 间质水肿型胰腺炎早期(<4周)液体积聚 | 无包膜，液体密度 |
| **ANC** | 坏死性胰腺炎早期(<4周)坏死物积聚 | 异质性，含坏死碎片 |
| **假性囊肿** | >4周后形成的包裹性液体 | 圆形，薄壁，均质液体 |
| **WON** | 包裹性坏死(>4周) | 厚壁，异质性内容物 |
| **感染性坏死** | 坏死组织感染 | 气体征(确诊) |

## 四、慢性胰腺炎

### 4.1 诊断标准 (IAP/APA 2024)

**1. 典型症状**
- 反复上腹部疼痛
- 胰腺外分泌功能不全(脂肪泻、营养不良)
- 胰腺内分泌功能不全(糖尿病)

**2. 影像学标准**

| 特征 | CT表现 | 评分 |
|------|--------|------|
| **钙化** | 胰腺实质/导管内钙化 | 轻/中/重度 |
| **胰管扩张** | 主胰管>3mm(老年人>4mm) | 轻/中/重度 |
| **胰管狭窄** | 上游扩张≥6mm | 确诊指标 |
| **胰腺萎缩** | 平均厚度<21mm(<14mm老年) | 轻/中/重度 |
| **假性囊肿** | >4cm或不吸收 | 并发症 |

### 4.2 钙化分级 (放射学严重程度评分)

| 程度 | 定义 |
|------|------|
| **轻度** | 细点状钙化<7个 |
| **中度** | 7-49个细点状钙化，或<7个粗大钙化 |
| **重度** | ≥50个细点状钙化，或≥7个粗大钙化 |

### 4.3 严重程度评估

| 指标 | 意义 |
|------|------|
| **腹痛频率** | 发作次数/月 |
| **脂肪泻** | 粪便弹性蛋白酶<200μg/g |
| **糖尿病** | 胰腺源性(需胰岛素) |
| **并发症** | 假性囊肿、脾静脉血栓、门脉高压 |

## 五、Pan-Agent系统实现建议

### 5.1 实性肿瘤模块

| 功能 | CT量化任务 | 输出标准 |
|------|------------|----------|
| **肿瘤分割** | 自动勾画肿瘤边界 | NIfTI mask + 体积ml |
| **血管评估** | SMA/SMV/CA/CHA/PV分割 | 接触角度计算 |
| **可切除性AI** | NCCN标准自动判断 | Resectable/BRPC/LAPC |
| **预后预测** | CA19-9 + 影像特征融合 | 生存期预测模型 |

### 5.2 囊性肿瘤模块

| 功能 | 量化指标 | 输出标准 |
|------|----------|----------|
| **囊性病变检测** | 自动识别囊性区域 | 位置+大小+囊壁厚度 |
| **壁结节检测** | 增强壁结节测量 | ≥5mm预警 |
| **分型AI** | SCN/MCN/IPMN/SPN | 多分类模型 |
| **恶性风险** | 综合评分系统 | 低/中/高风险 |

### 5.3 急性胰腺炎模块

| 功能 | 量化指标 | 输出标准 |
|------|----------|----------|
| **胰腺分割** | 正常/水肿/坏死组织 | 坏死百分比计算 |
| **积液分割** | APFC/ANC/WON自动识别 | 体积+类型 |
| **CTSI自动评分** | Balthazar+坏死评分 | 0-10分自动输出 |
| **严重程度AI** | MAP/MSAP/SAP/CAP | 分级+预后预测 |

### 5.4 慢性胰腺炎模块

| 功能 | 量化指标 | 输出标准 |
|------|----------|----------|
| **钙化检测** | 钙化数量+大小+分布 | 轻/中/重度分级 |
| **胰管分析** | 直径测量+狭窄定位 | 扩张/狭窄/结石 |
| **假性囊肿** | 检测+测量+随访对比 | 大小变化追踪 |
| **严重程度AI** | 综合影像+功能指标 | 分期+治疗建议 |
